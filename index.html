Modified language .txt files will be in /en. Replace your game language files with these to use your modifications.

.language files will be in /language_ext, which are just renamed .txt files. These can be loaded without replacing your default files when using BepInEx. Place them in BepInEx\plugins\SOME_FOLDER\
